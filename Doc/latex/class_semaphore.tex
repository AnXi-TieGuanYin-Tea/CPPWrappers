\-The semaphor is locked. 

\-When calling \hyperlink{class_semaphore_ab50da3ab3fdc7f56acc85cbdf248c98d}{wait()}, if the semaphore's value is zero, the calling thread will block until the lock is acquired or until the call is interrupted by a signal. \-Alternatively, the trywait() function will fail if the semaphore is already locked, rather than blocking on the semaphore. \begin{DoxyReturn}{\-Returns}
\-If successful (the lock was acquired), \hyperlink{class_semaphore_ab50da3ab3fdc7f56acc85cbdf248c98d}{wait()} and trywait() will return 0. \-Otherwise, -\/1 is returned and errno is set, and the state of the semaphore is unchanged. 
\end{DoxyReturn}
