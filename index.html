CPPWrappers by bend

C++ wrappers for the pthread, socket and semaphore library

Overview

This library is a C++ wrapper for BSD sockets, pthread and semaphore. I've test it on MacOS, but it should be working on Linux Therer are 2 Libraries: System, that contains the Thread class and Semaphore class Socket, that contains the TcpSocket and TcpServer class You can found the online documentation here

Install

To compile and install the library go to the folder System or Socket and type make sudo make install You need to do this for both folders

Examples

Small examples are located in the Tests folder. Basicaly the libraries are easy to use, notice that for the Thread class you need to subclass it and override the run() method
