wait()
int Semaphore::wait ( )

The semaphor is locked.

When calling wait(), if the semaphore's value is zero, the calling thread will block until the lock is acquired or until the call is interrupted by a signal. Alternatively, the trywait() function will fail if the semaphore is already locked, rather than blocking on the semaphore.

Returns:
If successful (the lock was acquired), wait() and trywait() will return 0. Otherwise, -1 is returned and errno is set, and the state of the semaphore is unchanged.
